Domain Ecology exists to:

- define NDH’s multi‑domain ecosystem

- **Root Domains (RD‑5)**

These classes define the ecological hierarchy.

2. **Bridge (BD‑5)**  
3. **Crown (CD‑5)**  

This gradient mirrors:

- Stability Geometry Gradient

- Domain Resonance Gradient  
- Ethical Gradient

NDH uses the same ordering everywhere to prevent drift.

## **🜁 VII. Domain Ecology Rules**

NDH defines three rules:

### **Rule 1 — Root Precedes Bridge**  
BD‑5 cannot exist without RD‑5.

### **Rule 2 — Bridge Precedes Crown**  
CD‑5 cannot exist without BD‑5.

### **Rule 3 — Crown Cannot Modify Root**  
CD‑5 may influence BD‑5 but cannot alter RD‑5.

These rules ensure ecological coherence.
