# **NDH Developer Toolkit v2.0**  
### *The complete SDK suite for integrating NDH into engines, XR platforms, and applications.*

---

## ⭐ 1. Purpose  
The NDH Developer Toolkit v2.0 provides:

- SDKs  
- bindings  
- templates  
- debugging tools  
- accessibility overlays  
- sensory adapters  
- geometry visualizers  
- tensor inspectors  
- multi‑manifold tools  

It is the **developer‑facing layer** of NDH v2.0.

---

## ⭐ 2. Toolkit Structure  
The toolkit contains seven major components:

- **Unity SDK**  
- **Unreal SDK**  
- **Godot SDK**  
- **WebXR/OpenXR SDK**  
- **Mobile/Desktop SDK**  
- **NDH Debug Suite**  
- **NDH Accessibility Overlay Suite**

Each SDK is HRD‑validated and accessibility‑anchored.
