## ⭐ 4. Tensor → Sensory Mapping  
Tensors drive sensory behavior:

### **Curvature → Visual Tone**
- smooth curvature → soft gradients  
- sharp curvature → high‑contrast tone  
- Dark Mode → inverted curvature mapping  

### **Stress → Haptic Intensity**
- low stress → gentle pulses  
- medium stress → rhythmic pulses  
- high stress → damped pulses (accessibility safety)  

### **Resonance → Audio Harmonics**
- low resonance → warm harmonics  
- medium resonance → layered harmonics  
- high resonance → filtered harmonics (HRD safety)  

### **Fold → Sensory Transitions**
- fold events → cross‑fade transitions  
- fold collapse → sensoryMode fallback  

### **Invariant Tensor → Stability Cues**
- stable invariants → ambient cues  
- unstable invariants → accessibility‑safe alerts
