# 🌌 **NDH‑AMS Badge Set Overview v1.1**  
### *The complete three‑tier mastery structure of the NDH cognitive manifold.*

The NDH‑AMS badge system is conceptual, not regulatory.  
It recognizes **mastery**, not legal authority.  
It has **three tiers**, each representing a deeper relationship with the manifold.

---

## ⭐ **Tier 1 — Practitioner Badge**  
**Role:** Applies NDH‑AMS  
**Focus:** Using the manifold in real conceptual scenarios  
**Core Skills:**  
- axis fluency  
- threshold recognition  
- attachment mapping  
- transition logic  
- emergent scenario analysis  

Explore: **Practitioner Badge**

---

## ⭐ **Tier 2 — Instructor Badge**  
**Role:** Teaches NDH‑AMS  
**Focus:** Guiding learners safely and coherently  
**Core Skills:**  
- teaching axes  
- teaching attachment geometry  
- teaching thresholds  
- running exercises  
- evaluating learners  
- maintaining conceptual safety  

Explore: **Instructor Badge**

---

## ⭐ **Tier 3 — Architect Badge**  
**Role:** Extends NDH‑AMS  
**Focus:** Designing, refining, and evolving the manifold  
**Core Skills:**  
- axis engineering  
- attachment architecture  
- operator/basin design  
- threshold system expansion  
- emergent dynamics modeling  
- documentation architecture  
- conceptual safety stewardship  

Explore: **Architect Badge**

---

## ⭐ **Progression Structure**

```
Practitioner  →  Instructor  →  Architect
   (Use)          (Teach)         (Design)
```

Each tier builds on the previous:

- Practitioner: “I can use NDH‑AMS.”  
- Instructor: “I can teach NDH‑AMS.”  
- Architect: “I can shape NDH‑AMS.”

## ⭐ **Badge Set Philosophy**  
The NDH‑AMS badge system is built on three principles:

- **Non‑hierarchical mastery** — tiers represent roles, not superiority  
- **Conceptual safety** — NDH‑AMS remains non‑diagnostic, non‑clinical, non‑political  
- **Structural coherence** — all extensions must preserve manifold geometry
