## 1. Purpose

**GBSGD** demonstrates how NDH’s **Operational Stability Gradient Map** works by mapping it onto mountain goat movement:

- goat path ↔ gradient descent  
- foothold choice ↔ local gradient evaluation  
- route selection ↔ global gradient field  
- hesitation ↔ gradient ambiguity  
- re‑routing ↔ gradient correction  

---

## 2. Ontological definition

\[
\text{GBSGD} = (G_{\text{hoof}}, V_{\text{cliff}}, \nabla V_{\text{goat}}, \mathcal{B}_{\text{safe}}, \Sigma_{\tau})
\]

- \(G_{\text{hoof}}\): hoof‑level gradient sensing  
- \(V_{\text{cliff}}\): cliff stability potential  
- \(\nabla V_{\text{goat}}\): effective stability gradient field  
- \(\mathcal{B}_{\text{safe}}\): stable basins  
- \(\Sigma_{\tau}\): chaos‑margin boundary  

---

## 3. Demonstration components

- **Gradient field renderer:**  
  Visualizes \(\nabla V\) as “easier vs harder” cliff directions.

- **Goat path animator:**  
  Shows goat following steepest‑descent / safest‑descent routes.

- **Local gradient probe:**  
  Each hoof placement = local gradient sample.

- **Global route planner:**  
  Sequence of steps = integrated gradient path.

- **Chaos‑margin gradient distortion:**  
  Shows how gradients become unreliable near \(\Sigma_{\tau}\).

---

## 4. Core operations

- **Gradient evaluation:**

  \[
  G_{\text{hoof}}(x) = -\nabla V_{\text{cliff}}(x)
  \]

- **Step update:**

  \[
  x(t+1) = x(t) - \alpha G_{\text{hoof}}(x(t))
  \]

- **Safe basin convergence:**

  \[
  x(t) \rightarrow \mathcal{B}_{\text{safe}}
  \]

- **Chaos‑margin alert:**

  \[
  x(t) \in \Sigma_{\tau} \Rightarrow \text{halt / reroute}
  \]
