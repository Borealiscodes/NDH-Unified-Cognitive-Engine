### 3. **Creates the NDH System Runtime Loop**  
This is the unified execution cycle.

### 4. **Creates the NDH System Teaching Loop**

### 5. **Creates the NDH System Creative Loop**  
This is the unified creative interface.

COS is the **final organ**.
