## ⭐ 2. Purpose  
The Metadata Backfill Execution (MBE) is the SDL‑driven process that:

- scans all NDH layers  
- detects missing metadata blocks  
- generates metadata blocks  
- validates metadata blocks  
- enforces invariants  
- aligns governance levels  
- ensures accessibility compliance  
- ensures neutrality compliance  
- ensures poverty‑safe compliance  
- integrates runtime markers  
- maintains canonical architectural order  

This is the **final coherence operation** of NDH v3.0.

  <version>3.0</version>
