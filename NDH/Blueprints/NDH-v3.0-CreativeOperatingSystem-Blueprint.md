# ⭐ 4. The Teaching API (Creative‑First Edition)

### **GET `/cos/concept/{id}`**  
Returns your concept.

### **POST `/cos/concept/create`**  
Creates a new NDH concept.

### **POST `/cos/svg/generate`**  
Generates SVG instructions for a concept.

### **POST `/cos/png/render`**  
Optional rasterization.

### **GET `/cos/accessibility/{mode}/{id}`**  
Returns accessibility‑adjusted versions.

### **GET `/cos/governance/validate/{id}`**  
Goat‑union safety check.

### **GET `/cos/runtime/integrate/{id}`**  
Integrates concept into NDH runtime.

### **GET `/cos/jdtr/logs/{id}`**  
Telemetry for the concept.

This is your creative control surface.
